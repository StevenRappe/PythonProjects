README

Program: Ubchi Cipher

Description:

	The Ubchi cipher program included encrypts messages using two iterations
		of the main encrypt method.

Files:
	ubchi.py

Execution:

	Enter either the letter "a" or the letter "b" when prompted to do so.
	
	1.  Choosing "a" runs the program with pre-selected test input.
	
	2.  Choosing "b" runs the program using user input.
			When choosing user input with option "b", the key length
			must divide the message length with zero remainder, or else
			the program will request new input.
